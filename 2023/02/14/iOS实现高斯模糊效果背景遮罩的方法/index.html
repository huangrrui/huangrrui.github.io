iOS实现高斯模糊效果背景遮罩的方法

Posted on 2023-02-14
  1. 使用原声的 UIEffectView 及 UIBlurEffect，优点是代码简单，缺点是自定义性较差，只有系统提供的几种固定效果，可能不满足设计需求，代码与效果如下
let effect_view = UIVisualEffectView(frame: self.view.bounds)
effect_view.effect = UIBlurEffect(style: .light)
self.view.addSubview(effect_view)

  1. 先将所需模糊的背景进行截图，一般截取整个屏幕，然后使用 CIFilter 进行高斯模糊，随后伪装成背景盖在原来的界面上即可，优点是模糊度可以自定义，缺点是由于是由于是假背景，不像原生那样是带透明度的，所以背景的一变化无法显示出来。代码与效果如下
/// 截屏
func screen_shot() -> UIImage? {
    guard let window = (UIApplication.shared.delegate as? AppDelegate)?.window else { return nil }
    UIGraphicsBeginImageContextWithOptions(UIScreen.main.bounds.size, false, UIScreen.main.scale)
    
    guard let context = UIGraphicsGetCurrentContext() else { return nil }
    window.layer.render(in: context)
    let image = UIGraphicsGetImageFromCurrentImageContext()
    UIGraphicsEndImageContext()
    return image
}

/// 模糊处理
func blur_image(image: UIImage) -> UIImage? {
    let context = CIContext(options: nil)
    guard let ciImage = CIImage(image: image) else { return nil }
    guard let filter = CIFilter(name: "CIGaussianBlur") else { return nil }
    filter.setValue(ciImage, forKey: kCIInputImageKey)
    // 设置模糊程度
    filter.setValue(NSNumber(floatLiteral: 10), forKey: "inputRadius")
    guard let result = filter.value(forKey: kCIOutputImageKey) as? CIImage else { return nil }
    guard let outImage = context.createCGImage(result, from: ciImage.extent) else { return nil }
    let blurImage = UIImage(cgImage: outImage)
    return blurImage
}
