    Swift基本操作符

    Posted on 2018-07-27

    Basic Operators （基本操作符）

    Terminology （术语）
    • Unary Operators （一元操作符, 例如-，! 等）
    • Binary Operators （二元操作符，例如+，-，*，/ 等）
    • Ternary Operators （三元操作符，例如?）
    Assignment Operator（赋值操作符）=
    let b = 10
var a = 5
a = b
let (x, y) = (1, 2)

    不同于 C 和 C++，他不会返回任何值，所以在下面的例子中会报错，很好的防止了手误导致错误

    if x = y { // Error
}
    Arithmetic Operators （算术操作符）

    不同于 C 或 C++，Swift 不允许溢出

    let a: UInt8 = 1 + 2
let b: UInt8 = 255
print(a + b) // Error，溢出了
print(a &+ b) // 输出 2
    Remainder Operator（取余操作符）
    9 % 4 = 1
9 % -4 = 1
-9 % 4 = -1
-9 % -4 = -1
    Unary Minus Operator （一元减号操作符）
    let three = 3
let minusThree = -three // minusThree = -3
let plusThree = -minusThree // plusThree = 3
    Unary Plus Operator （一元加号操作符）

    没啥用，主要功能是对齐格式，例如

    let minusSix = -6
let  plusSix = +6

    Compound Assignment Operator （复合赋值操作符）

    var a = 1
a += 2 // a = 3
a *= 2 // a = 6
a /= 3 // a = 2
// 它不会返回任何值，所以不能写成下面的样子
let b = a += 2 // Error

    Comparison Operator （比较操作符）

    比较语句会返回一个布尔值

    1 == 1   // true
2 != 1   // true
2 > 1    // true
1 < 2    // true
1 >= 1   // true
2 <= 1   // false

    元组的比较
    仅当两个元组类型完全一样的时候才可以比较，例如

    let a = (1, 2)
let b = (2, 3)
a == b // 可以比较

let c = (2, "2")
a == c // 不能比较，类型不一致

let d = (1, 2, 3)
a == d // 不能比较，数量不一致

    满足数量与类型的条件后，开始从第一个值开始一次比较，如果每一个值都相等，那么这两个元组就是相等的

    let a = (1, 2, "3")
let b = (1, 2, "3")
a == b // True

    当遇到两个不相等的值的时候，比较这两值，将结果作为这两个元组的比较结果返回，并停止后续的比较

    (1, "zebra") < (2, "apple")   // true because 1 is less than 2; "zebra" and "apple" aren't compared
(3, "apple") < (3, "bird")    // true because 3 is equal to 3, and "apple" is less than "bird"
(4, "dog") == (4, "dog")      // true because 4 is equal to 4, and "dog" is equal to "dog”

    假如元组中包含了不能够比较类型，例如 Bool，则这两个元组是不能进行大小比较的，但是可以进行相等比较

    let a = (true, true)
let b = (false, false)
a < b // Error
a == b // false

    Ternary Conditional Operator （三元条件操作符）

    ?操作符

    let a = 1 > 2 ? 1 : 2 // a = 2

    Nil-Coalescing Operator （空合运算符）

    var a: String?
var b = a ?? "a is nil" // b = "a is nil"
a = "123"
b = a ?? "a is nil" // b = "123"

    Range Operator （区间运算符）

    Closed Range Operator （闭区间范围运算符）
    for index in 1...5 {
    print(index)
}
// 1
// 2
// 3
// 4
// 5
    Half-Open Range Operator （半开区间运算符）
    for i in 1..<5 {
    print(i)
}
// 1
// 2
// 3
// 4
    One-Sided Ranges （单边区间运算符）

    1、指定了起始点

    for i in 1... {
    print(i)
}
// 1
// 2
// 3
// 4
// ......

let list = ["1", "2", "3", "4"]
for str in list[2...] {
    print(str)
}
// 3
// 4

    2、指定结束点，

    let list = ["1", "2", "3", "4"]
for str in list[...2] {
    print(str)
}
// 1
// 2
// 3

// 作为序列时，不可打印
for i in ...1 {
    print(i) // Error
}
let range = ...5
range.contains(7) // false
range.contains(4) // true
range.contains(-1) // true

    Logical Operator （逻辑操作符）

    Logical NOT Operator （逻辑非）
    let allowedEntry = false
if !allowedEntry {
    print("ACCESS DENIED")
}
// Prints "ACCESS DENIED"
    Logical AND Operator （逻辑与）
    let enteredDoorCode = true
let passedRetinaScan = false
if enteredDoorCode && passedRetinaScan {
    print("Welcome!")
} else {
    print("ACCESS DENIED")
}
// Prints "ACCESS DENIED”
    Logical OR Operator （逻辑或）
    let hasDoorKey = false
let knowsOverridePassword = true
if hasDoorKey || knowsOverridePassword {
    print("Welcome!")
} else {
    print("ACCESS DENIED")
}
// Prints "Welcome!"
    Combining Logical Operator （组合逻辑）
    if enteredDoorCode && passedRetinaScan || hasDoorKey || knowsOverridePassword {
    print("Welcome!")
} else {
    print("ACCESS DENIED")
}
// Prints "Welcome!"

    Swift 默认顺序是从左到右的，但是为了方便阅读，最好加上括号显式指定顺序

    Explicit Parentheses （显式括号）
    if (enteredDoorCode && passedRetinaScan) || hasDoorKey || knowsOverridePassword {
    print("Welcome!")
} else {
    print("ACCESS DENIED")
}
// Prints "Welcome!"
